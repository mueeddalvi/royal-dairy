## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Fluid spacing using CSS `clamp()` functions for responsive sections (e.g., `clamp(2rem, 5vw, 6rem)`).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #020B1E | Deep luxury navy background, rich ambient blocks |
| Secondary (30%) | #FFFFFF | Pristine white text, high contrast headers |
| Accent (10%) | #D4AF37 | Metallic gold for preloader, CTAs, luxury highlights |
| Platinum/Silver | #D8D9DA | Borders, secondary labels, divider lines |
| Destructive | #8B0000 | Deep crimson for alert states, form errors |

Accent reserved for: Primary interactive buttons, preloader active state, and brand logo marks.
